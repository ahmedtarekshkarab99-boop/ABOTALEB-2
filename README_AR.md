# نظام إدارة مطحنة القهوة
# Coffee Mill ERP System - Arabic-First Desktop Application

نظام متكامل لإدارة عمليات مطحنة القهوة (التحميص والطحن) مع دعم كامل للغة العربية والتخطيط من اليمين إلى اليسار.

## المميزات الرئيسية

### الموديولات
- **إدارة المخزن**: تتبع المواد الخام والمنتجات النهائية
- **إدارة الموردين**: تسجيل والموردين وتتبع الشراء
- **إدارة العملاء**: تسجيل العملاء وتتبع المبيعات
- **إدارة العاملين**: الحضور والرواتب وإدارة الموارد البشرية
- **إدارة الإنتاج**: تتبع دفعات التحميص والطحن
- **تتبع المصروفات**: تسجيل وتصنيف المصروفات
- **المبيعات والمشتريات**: فواتير البيع والشراء والدفعات

### المميزات التقنية
- ✅ **عربي بنسبة 100%**: جميع عناصر الواجهة باللغة العربية
- ✅ **RTL كامل**: تخطيط من اليمين إلى اليسار على كامل النظام
- ✅ **Offline-First**: يعمل بشكل مستقل بدون اتصال إنترنت
- ✅ **قاعدة بيانات محلية**: SQLite محفوظة في ملف واحد
- ✅ **بدون تثبيت**: يعمل مباشرة في المتصفح
- ✅ **سريع وخفيف**: أداء ممتاز مع استهلاك موارد قليل

## المتطلبات

- Node.js 18+ 
- npm أو yarn

## التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
git clone <repo-url>
cd coffee-mill-erp
```

### 2. تثبيت الحزم
```bash
npm install
```

### 3. التطوير
```bash
npm run dev
```
سيفتح التطبيق في `http://localhost:5173`

### 4. البناء للإنتاج
```bash
npm run build
```
سيتم إنشاء مجلد `dist` بالملفات الجاهزة للنشر.

## الملفات المهمة

### لغة النظام
- `src/locales/ar.json`: جميع النصوص والعبارات
- `src/lib/i18n.ts`: نظام الترجمة

### قاعدة البيانات
- `src/db/client.ts`: اتصال قاعدة البيانات
- `src/db/schema/`: تعريفات الجداول
- `src/db/repositories/`: دوال الوصول للبيانات

## أوامر مفيدة

```bash
# التطوير مع الـ Hot Reload
npm run dev

# البناء للإنتاج
npm run build
```
